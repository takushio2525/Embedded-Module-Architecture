\section{モジュールリファレンス}\label{sec:module-ref}

本章では、サンプル実装（\texttt{sample/}）で提供される各モジュールのAPI仕様を記述する。
各モジュールは第\ref{sec:api}節で定義した\texttt{IModule}インターフェースに準拠し、
Config構造体・Data構造体・ライフサイクルメソッドの統一パターンに従う。

\begin{notebox}[本章の位置づけ]
第\ref{sec:api}節がフレームワークの設計パターンとルールを定義するのに対し、
本章は各モジュールの具体的なAPI仕様（Config/Dataメンバ、メソッド動作、制約事項）を記述する。
ライブラリリファレンスに相当する位置づけである。
\end{notebox}
