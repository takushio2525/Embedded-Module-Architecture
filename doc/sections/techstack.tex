\subsection{PlatformIO設定}

\begin{lstlisting}[basicstyle=\ttfamily\small, frame=single, caption={platformio.ini}]
[env:esp32-s3-cam-n16r8]
platform = espressif32
board = esp32-s3-devkitc-1
framework = arduino
monitor_speed = 115200

; --- N16 (16MB Flash) ---
board_upload.flash_size = 16MB
board_build.partitions = default_16MB.csv

; --- R8 (8MB PSRAM) ---
board_build.arduino.memory_type = qio_opi
build_flags =
    -D BOARD_HAS_PSRAM
    -D CORE_DEBUG_LEVEL=3
    -I include  ; lib/内の.cppからinclude/を参照可能にする
\end{lstlisting}

\begin{notebox}[\texttt{-I include}が必要な理由]
PlatformIOの標準設定では、\texttt{lib/}内のライブラリはプロジェクトの\texttt{include/}ディレクトリを
インクルードパスに含まない。
モジュールの\texttt{.cpp}ファイルが\texttt{ProjectConfig.h}を参照するため、
\texttt{build\_flags}に\texttt{-I include}を追加してパスを通す必要がある。
\end{notebox}
